# Portfolio Project Requirements

## Overview
A professional portfolio website for S.Roobin, showcasing skills and experience as a Java Full Stack Developer.

## Features
1. Responsive design that works on all devices
2. Modern and clean UI
3. Sections for:
   - Hero/Introduction
   - About Me
   - Skills
   - Projects
   - Experience
   - Contact Information

## Technical Stack
- HTML5
- CSS3 (with Flexbox/Grid)
- JavaScript (Vanilla)
- Font Awesome for icons
- Google Fonts for typography

## Project Structure
```
Portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── images/
└── README.md
```

## Design Guidelines
- Color Scheme: Professional and modern
- Typography: Clean and readable
- Layout: Single page with smooth scrolling
- Animations: Subtle and professional
- move the profile picture to the right of the highlight text
