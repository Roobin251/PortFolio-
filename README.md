# Personal Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. This portfolio showcases your skills, projects, and experience as a Java Full Stack Developer.

## Features

- Responsive design that works on all devices
- Modern and clean UI with smooth animations
- Sections for:
  - Hero/Introduction
  - About Me
  - Skills
  - Projects
  - Experience
  - Contact Information
- Mobile-friendly navigation
- Smooth scrolling
- Contact form
- Social media links

## Getting Started

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. Customize the content to match your personal information

## Customization

### Personal Information
1. Open `index.html`
2. Update the following sections with your information:
   - Hero section: Your name and title
   - About section: Your bio and statistics
   - Skills section: Your technical skills
   - Projects section: Your portfolio projects
   - Experience section: Your work history
   - Contact section: Your contact information and social media links

### Styling
1. Open `css/style.css`
2. You can customize:
   - Colors: Update the CSS variables in the `:root` section
   - Typography: Change the Google Fonts link in `index.html`
   - Layout: Modify the grid and flexbox properties
   - Animations: Adjust the animation timings and effects

### Images
1. Add your images to the `images` folder
2. Update image paths in `index.html`
3. Recommended image sizes:
   - Project thumbnails: 800x600px
   - Profile picture: 400x400px
